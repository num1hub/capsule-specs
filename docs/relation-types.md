# Canonical Relation Types

Capsule links use a fixed set of relation types so the graph remains computable.

## Canonical relation types

- `supports`
  Strengthens the target with evidence or argument.
- `contradicts`
  Directly opposes the target and raises contradiction pressure.
- `extends`
  Adds depth or scope without replacing the original meaning.
- `derived_from`
  Indicates direct lineage or byproduct status.
- `depends_on`
  Marks a strict logical or execution dependency.
- `references`
  Creates a neutral contextual pointer.
- `duplicates`
  Indicates materially identical content.
- `implements`
  Marks practical realization of a theory, blueprint, or interface.
- `part_of`
  Expresses strict compositional membership in a parent or hub.

## Why this matters

Canonical relation types prevent graph drift. Once arbitrary edge labels appear, traversal logic and topology-aware automation become much harder to trust.
