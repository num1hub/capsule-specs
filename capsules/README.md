# Capsules

This directory contains a curated raw capsule set used as public source material for the repository.

These files are included because they are high-signal and safe to expose publicly:

- the 5-Element Law
- the 16 Validation Gates
- canonical relation types
- the schema capsule
- the confidence-vector law surface
- the version-lineage protocol
- the structural subtype semantics for `atomic` and `hub`

They are curated because they reinforce already-published public contracts:

- capsule shape and sealing
- confidence-vector semantics behind `G15`
- stable subtype semantics used by schemas, examples, and package projections
- immutable version lineage without destructive edits

They are intentionally not a full vault export.
